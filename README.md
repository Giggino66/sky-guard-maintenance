
# SkyGuard Maintenance Pro

Sistema avanzato per la gestione delle scadenze aeronautiche e inventario componenti.

## Come eseguire localmente
1. Assicurati di avere [Node.js](https://nodejs.org/) installato.
2. Clona il repository.
3. Esegui `npm install`.
4. Crea un file `.env` e aggiungi `API_KEY=tua_chiave_gemini`.
5. Esegui `npm run dev`.
